# Petstore API Automation Framework

## Overview
This project automates the testing of the **Swagger Petstore API** using:
- **Rest Assured** for API interaction
- **TestNG** for test orchestration
- **Maven** for build automation
- **Allure** for detailed reporting

## Tech Stack
| Component | Tool |
|------------|------|
| Programming Language | Java |
| API Testing Library | Rest Assured |
| Test Runner | TestNG |
| Build Tool | Maven |
| Reporting | Allure |
